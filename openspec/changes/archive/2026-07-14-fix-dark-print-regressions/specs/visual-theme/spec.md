## MODIFIED Requirements

### Requirement: Dark theme uses distinct color palette
The dark theme SHALL use a slate-based palette with sufficient contrast: surface `#1e293b`, surface-alt `#334155`, border `#475569`, text `#f1f5f9`, text-muted `#94a3b8`.

#### Scenario: Dark mode uses slate palette
- **WHEN** dark mode is active
- **THEN** colors use the slate palette with clear contrast between surface, border, and text
